# Luogu-Plus

一个针对洛谷进行优化的浏览器插件。

## 计划

- ✅ 替换主页板块
- ✅ 去除广告
- ❌ AI解题 (开发过, 因为bug多加上洛谷规定导致被ban)
- ❌ 全局设置 (开发中)

## 安装环境

- 你的浏览器需基于Chromium内核
- 建议浏览器内核≥100
